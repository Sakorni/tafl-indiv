# Задание 10

Чтобы программа могла прочитать грамматику из файла используйте следующий синтаксис:

`(N->b(|b)*\n)+`

Где b является конкатенацией терминалов и нетерминалов 

т.е. `(a+b)+`, где a - Терминал, b - нетерминал
